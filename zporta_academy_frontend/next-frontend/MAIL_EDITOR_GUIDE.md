# Mail Magazine Editor - User Guide

## Quick Start

### Creating a Professional Email

1. Navigate to Teacher Mail Magazine section
2. Click **"Compose New Magazine"**
3. Fill in:
   - **Title**: Internal name for your mail magazine
   - **Subject**: Email subject line (what recipients see)
   - **Message Body**: Use the rich text editor (see below)

## Platform Linking

### How to Link Content

1. Click the appropriate icon:

   - **ðŸ“š Lesson** for lessons
   - **ðŸŽ“ Course** for courses
   - **ðŸ“‹ Quiz** for quizzes

2. **Search Modal Opens**:

   - Type at least 2 characters to search
   - Results appear automatically
   - Click on an item to select it

3. **Insert Link**:
   - Selected item is highlighted in yellow
   - Click "Insert Link" button
   - Link appears in editor with golden color

### Link Format

Links are created with:

- Golden color (#ffb703)
- Underline style
- Direct URL to content
- Visible title text

**Example**:

```
Check out this lesson: [Introduction to Python]
```

## Templates

### Using Pre-Built Templates

1. Click **"Templates"** tab
2. Browse 4 available templates:

   - **Welcome Email**: For new students
   - **Content Update**: Announce new materials
   - **Study Reminder**: Encourage engagement
   - **Achievement**: Celebrate milestones

3. Click **"Use This Template"**
4. Editor loads with template content
5. Customize text, links, and images
6. Save as new magazine

### Template Structure

Each template includes:

- Professional greeting
- Content body with placeholders
- Call-to-action
- Closing signature

**Tip**: Replace `[Student Name]`, `[Course Name]`, etc. with actual content

## Image Upload

### Adding Images

1. Click **ðŸ–¼ Image** button in toolbar
2. **Choose File** from your computer
3. Wait for upload (progress not visible)
4. Image appears in editor

### Image Best Practices

- **Size**: Keep under 500KB for email compatibility
- **Format**: JPEG or PNG recommended
- **Dimensions**: Max 600px wide for email clients
- **Alt Text**: Currently not supported (future feature)

### Troubleshooting Images

- **No image?** Check file size (<5MB server limit)
- **Slow upload?** Large files take time
- **Image not showing?** Verify internet connection

## Email Composition Tips

### Subject Line Best Practices

- Keep under 50 characters
- Use action words: "Discover", "Learn", "Join"
- Avoid spam triggers: "FREE", "ACT NOW"
- Personalize when possible

### Body Content Guidelines

1. **Opening**: Warm, personal greeting
2. **Main Content**:
   - Use short paragraphs (2-3 sentences)
   - Include platform links to relevant content
   - Add images for visual interest
3. **Call-to-Action**: Clear next step
4. **Closing**: Professional sign-off

## Frequency Settings

### Schedule Options

- **One Time**: Send once immediately or at scheduled time
- **Daily**: Automated daily delivery
- **Weekly**: Weekly recurring email
- **Monthly**: Monthly newsletter

### Scheduled Send

1. Select frequency
2. Set **Scheduled Send Time**:
   - Use date/time picker
   - All times in your local timezone
3. Check **"Keep this mail magazine active"**

**Note**: Inactive magazines won't send automatically

## Preview & Testing

### Before Sending

1. Click **"View Details"** on saved magazine
2. Review:

   - Subject line
   - HTML body preview
   - Frequency settings
   - Recipient count

3. Send test email:
   - Click **"Send Email Now"**
   - Confirm sending
   - Check your own inbox

### HTML Preview

- Preview shows formatted content
- Links are clickable
- Images display (if uploaded)
- Styling matches email appearance

## Editing Existing Emails

### Modify Saved Magazine

1. Find magazine in list view
2. Click **"Edit"** button
3. Content loads into editor automatically
4. Make changes using toolbar
5. Click **"Update Magazine"**

**Note**: Content loads with 100ms delay for proper rendering

### Cancel Editing

- Click **"Cancel"** button
- Returns to list view
- Editor clears automatically
- No changes saved

## Troubleshooting

### Editor Issues

**Problem**: Editor won't load content  
**Solution**: Refresh page, try editing again

**Problem**: Formatting not saving  
**Solution**: Ensure HTML content is generated (check preview)

**Problem**: Links not working  
**Solution**: Verify URL includes `https://` or `http://`

### Content Issues

**Problem**: Platform search returns no results  
**Solution**: Check spelling, try broader search terms

**Problem**: Image won't upload  
**Solution**: Verify file size (<5MB) and format (JPEG/PNG)

**Problem**: Template won't load  
**Solution**: Switch to Compose view, try again

## Best Practices Summary

### Content Strategy

1. **Consistency**: Regular sending schedule
2. **Value**: Always provide useful content or links
3. **Engagement**: Include platform links to lessons
4. **Brevity**: Keep emails concise and scannable
5. **Testing**: Send test to yourself first

### Technical Tips

1. **Save Often**: Click save after major edits
2. **Preview First**: Always check detail view before sending
3. **Test Links**: Verify platform links work
4. **Image Optimization**: Compress images before upload
5. **Mobile-Friendly**: Keep content simple for mobile readers

### Email Etiquette

1. **Subject Lines**: Clear and relevant
2. **Greeting**: Personal but professional
3. **Body**: Focused on one main topic
4. **CTA**: Single, clear call-to-action
5. **Signature**: Consistent sign-off
